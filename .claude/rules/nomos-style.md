# N0mosAi 输出规范

本文档定义 N0mosAi 系统的输出格式规范，在每次会话开始时自动注入。


### 1.2 框线字符

| 元素 | 字符 | 示例 |
|------|------|------|
| 水平线 | `─` | `──────` |
| 垂直线 | `│` | `│` |
| 角 | `┌ ┐ └ ┘` | `┌────┐` |
| 连接 | `├ ┤ ┬ ┴ ┼` | `├────` |
| 箭头 | `▼ ► ◀ ▲` | `▼` `──►` |

### 1.3 箭头

```
简单: → ← ↑ ↓

```

## 2. Emoji 状态图标库

### 2.1 状态与结果

| Emoji | 含义 | 使用场景 |
|:---:|------|---------|
| ✅ | 完成/通过 | 任务完成、测试通过 |
| ❌ | 失败/错误 | 任务失败、测试失败 |
| ⏳ | 进行中/等待 | 任务进行中 |
| ⏸️ | 暂停/阻塞 | 等待依赖 |
| ⚠️ | 警告/注意 | 需要注意 |
| 🚫 | 不可用/禁用 | 功能禁用 |

### 2.2 角色与身份

| Emoji | 含义 |
|:---:|------|
| 👤 | 用户/人类 |
| 🤖 | Agent/AI |
| 🔍 | 审查者 |

### 2.3 文件与操作

| Emoji | 含义 |
|:---:|------|
| 📄 | 文档/文件 |
| 📝 | 笔记/标注 |
| 📋 | 清单/列表 |
| ➕ | 添加 |
| ➖ | 删除 |
| ✏️ | 编辑 |
| 🔧 | 配置 |

### 2.4 Git 与版本

| Emoji | 含义 |
|:---:|------|
| 📦 | 提交 |
| 🔀 | 分支 |
| 🏷️ | 标签 |
| ⏪ | 回滚 |
| 📤 | 推送 |
| 📥 | 拉取 |

### 2.5 颜色状态

| 状态 | Emoji | 文字描述 |
|------|:---:|---------|
| 成功 | 🟢 | `[GREEN]` `[OK]` |
| 警告 | 🟡 | `[YELLOW]` `[WARN]` |
| 错误 | 🔴 | `[RED]` `[ERROR]` |
| 信息 | 🔵 | `[BLUE]` `[INFO]` |
| 待处理 | 🟠 | `[ORANGE]` `[PENDING]` |

### 2.6 严重程度

| 严重程度 | Emoji | 标记格式 | 阻塞行为 |
|---------|:---:|---------|---------|
| 致命 | 🚨 | `[CRITICAL]` | 阻塞 + 可能 Revert |
| 严重 | ⚠️ | `[MAJOR]` | 阻塞 |
| 次要 | 📝 | `[MINOR]` | 不阻塞 |
| 建议 | 💡 | `[SUGGEST]` | 不阻塞 |

## 3. 术语约定

| 术语 | 含义 |
|------|------|
| N0mosAi | 项目品牌名 |
| Nomos | 项目内部引用名 |
| nomos | CLI 命令前缀 |
| Agent | AI 智能体（如 Claude） |

| Phase | 阶段（Why-First/Research/Plan/Execute/Review） |

| 标注/RC | Review Comment，人类批注 |
